# MVP Scope

This MVP compares 28 European countries using four indicators:

- inflation_pressure
- housing_pressure
- poverty_pressure
- income_capacity

The first three are pressure indicators where lower values are generally better. Income capacity is different: higher values indicate stronger income capacity, and it is not a pressure metric.

The MVP includes:
- Country-level pressure and income capacity insights
- Relative ranking across indicators
- Two-country comparison logic
- Income capacity presented in PPS

The MVP does not claim to be a full relocation recommendation engine.
