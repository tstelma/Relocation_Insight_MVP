# Data Sources

- **Eurostat HICP**: Used for inflation_pressure.
- **Eurostat SILC**: Used for housing_pressure and poverty_pressure.
- **Eurostat ilc_di03**: Used for income_capacity.

Output files include:
- `data/clean/income_capacity_mvp_countries.csv`
- `data/clean/income_capacity_insights.csv`
- `data/clean/all_mvp_insights.csv`

The first three indicators are cost and social pressure metrics; income_capacity is a separate income-strength signal.
